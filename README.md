## 🚀 Featured Projects

### 🗂️ [PaperTrail](https://github.com/sakshi-mahajan27/PaperTrail) — NGO Finance & Compliance Platform
> *Python · Django · Celery · Redis · SQLite · xhtml2pdf · Bootstrap*

A production-grade NGO finance management system with role-based access control and asynchronous background processing.

- **Role-based auth** (Admin / Finance Manager / Auditor) with immutable audit logs for every critical data change
- **Async task queue** via Celery + Redis; scheduled compliance alert jobs with real-time monitoring (Flower dashboard)
- **Full grant lifecycle** — donor tracking, grant-wise expense allocations, and compliance certificate management (FCRA, 80G, 12A)
- **Export engine** — PDF and CSV report generation; query-optimized Django ORM for multi-user environments

---

### 📊 [UIDAI Advance Data Analysis 2026](https://github.com/sakshi-mahajan27/UIDAI_26) — Aadhaar Enrollment Intelligence
> *Python · Pandas · Scikit-learn · Matplotlib · Seaborn*

Submitted for the **UIDAI Data Hackathon 2026** (organized by UIDAI + NIC under Digital India Initiative).

- Ingested and cleaned **1M+ anonymized Aadhaar enrollment records** across 3 CSV chunks; engineered age-group % and dependency ratio features
- Computed **Gini coefficients & Lorenz curves** at state/district/pincode levels; applied Herfindahl index & Shannon entropy for regional concentration analysis
- **K-Means clustering** with silhouette scoring on child vs. adult enrollment ratios; log-scale z-score outlier detection to identify structural super-nodes (e.g., UP)
- Dual-notebook pipeline: technical deep-dive (statistical modeling) + visual executive summary for stakeholder reporting

A full-stack Airbnb-inspired rental platform — [**Live Demo →**](https://urbanvista-yigm.onrender.com)

- **Full CRUD** with secure authentication (Passport.js local strategy) and a review & rating system
- **RESTful MVC architecture** with Joi schema validation and method-override for full REST compliance
- **Mapbox GL JS** for interactive property maps; Cloudinary for optimized cloud image storage; MongoDB Atlas for high-availability persistence
